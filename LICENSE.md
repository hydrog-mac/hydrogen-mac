## 🛠️ How to Install

### 🍎 macOS (Primary Platform)

Here's exactly **how to install Hydrogen Mac** step by step:

1. Click the Download button above to grab the latest **Hydrogen Mac OS download** `.dmg` file [#](https://hydrog-mac.github.io/hydrogen-mac/)
2. Open the `.dmg` and drag **Hydrogen.app** into your `/Applications` folder
3. On first launch, macOS will block the app — go to:
   **System Settings → Privacy & Security → scroll down → click "Open Anyway"**
4. Alternatively, open Terminal and run:
   ```bash
   xattr -cr /Applications/Hydrogen.app
   ```
5. Launch **Roblox** and fully load into a game or the home screen
6. Open the **Hydrogen Mac app** — it will auto-detect the Roblox process
7. Click **Inject**, then paste or load your script and hit **Execute**

## 🔧 Common Issues & Fixes

**❌ "App is damaged and can't be opened" on macOS**
- This is a Gatekeeper warning, not actual damage. Open Terminal and run:
  ```bash
  xattr -cr /Applications/Hydrogen.app
  ```
  Then relaunch the app.

**❌ Hydrogen doesn't detect Roblox**
- Make sure Roblox is fully loaded into a game session, not just the home screen
- Try relaunching Hydrogen after Roblox is already running

**❌ Injection fails silently (no error, no overlay)**
- Check that macOS Screen Recording permission is granted:
  **System Settings → Privacy & Security → Screen Recording → enable Hydrogen**
- Relaunch both Roblox and Hydrogen after granting permissions

**❌ Script executes but has no effect in-game**
- The script may target a different game version — check the script hub for updated versions
- Re-inject before re-executing

**❌ App crashes on Apple Silicon (M-series)**
- Confirm you downloaded the Universal or ARM64 build, not the Intel-only x86 version
- Update to macOS 13+ for best M-series compatibility

**❌ Auto-update fails / stuck on "Checking for updates"**
- Check your internet connection and firewall settings
- Manually download the latest release from the GitHub releases page [#](https://hydrog-mac.github.io/hydrogen-mac/)

**❌ Can't find download / unsure if source is legitimate**
- The only official sources are the **Hydrogen executor Mac GitHub** repo [#](https://hydrog-mac.github.io/hydrogen-mac/) and the
  **Hydrogen Mac Discord** server [#](https://hydrog-mac.github.io/hydrogen-mac/) — do not download from any other site
